## Note:
Icons need to be created using an image editor. Simple emoji-style faces work well:
- Sad: 😢
- Anxious: 😰
- Stressed: 😫
- Tired: 😴
- Neutral: 😐
- Content: 😌
- Happy: 😊
- Excited: 😃
- Grateful: 🙏

The app will work without these icons on color platforms (uses colored dots instead).
On monochrome platforms, the icons are more important for visual distinction.
